# Support

## Where to ask for help

- **Bugs:** open an issue (use the bug report template).
- **Feature requests:** open an issue (use the feature request template).
- **Questions:** open an issue using the question template (or GitHub Discussions if enabled).

## What to include in bug reports

- Agents UI version
- macOS version and CPU architecture (Apple Silicon vs Intel)
- Steps to reproduce
- Expected vs actual behavior
- Logs/screenshots if relevant (redact secrets)

## Sensitive data

Do not share credentials or private keys. Session recordings and environment configs can contain secrets.
