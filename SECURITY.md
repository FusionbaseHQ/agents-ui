# Security Policy

Agents UI is a local desktop application that runs shells and agent CLIs with your user permissions. Treat any agent output as untrusted.

## Supported versions

Only the latest released version is supported with security updates.

## Reporting a vulnerability

Please report security issues privately.

Preferred:
- Use GitHub “Report a vulnerability” (Security tab) if enabled for the repository.

Alternative:
- Email the maintainers at: **info@fusionbase.com**

Include:
- affected version(s),
- OS + CPU architecture,
- clear reproduction steps or PoC,
- impact assessment (what an attacker gains),
- any relevant logs/screenshots (redact secrets).

## Disclosure process

- We’ll acknowledge receipt as soon as possible.
- We’ll investigate, prepare a fix, and coordinate a release.
- Please avoid public disclosure until a patched release is available.
